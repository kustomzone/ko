# about ko

ko is an editor for Mac OS X
    
## main features

- fast
- minimap
- terminal
- file search
- multicursors
- autocomplete
- syntax highlighting

### missing stuff

- no plugin system
- no git integration
- no bracket matching
- limited language support

#### notes 

- completely undocumented
- pre-release in active development
- use at your own risk!
